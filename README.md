# Gold DCA cBot

Systematic XAUUSD dip-buying strategy for cTrader.

## Setup
1. Install cTrader desktop (Windows)
2. `conda env create -f environment.yml`
3. `conda activate gold-dca`
4. Open repo in VS Code via WSL

## Structure
- `src/GoldDcaBot/` — C# cBot source
- `tests/` — unit tests
- `analysis/` — Python backtest analysis
- `docs/` — strategy notes
